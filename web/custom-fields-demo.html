How to Use Custom Fields

1. Admin Panel: Go to the admin panel and click on "Custom Fields" to manage all form fields.

2. Add Fields: Click "Add New Field" to create custom fields for any section.

3. Field Types: Support for text, email, phone, number, textarea, dropdown, radio, checkbox, date, file upload, and more.

4. Validation: Set required fields, min/max length, patterns, and custom validation rules.

5. Placement: Choose which section the field appears in and set display order.

6. Styling: Customize width, CSS classes, placeholders, and help text.

7. Dynamic Loading: Fields are automatically loaded and rendered in forms without code changes.
